# AGENTS.md

本项目为大学课程设计项目。

目标：

实现一个可视化电脑迷宫鼠系统。

开发原则：

代码必须简单。

代码必须容易答辩。

代码必须容易阅读。

禁止过度设计。

禁止复杂设计模式。

禁止无意义抽象。

---

优先级：

功能正确

> 

动画展示

> 

代码规范

> 

界面美观

> 

高级架构

---

必须使用：

Java

Swing

Stack

Queue

二维数组

---

必须实现：

Prim迷宫生成

DFS寻路

BFS最短路径

实时动画

日志系统

状态面板

---

动画要求：

逐步展示搜索过程

逐步展示回溯过程

逐步展示最终路径

支持暂停

支持继续

支持速度调节

---

答辩要求：

每个类职责明确

每个算法独立封装

每个模块可单独讲解

代码应适合课程答辩

避免复杂框架

避免设计模式堆砌

避免难以解释的实现

---

生成代码时：

先设计

再实现

最后测试

确保可以编译运行
